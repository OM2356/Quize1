## 🚀 Overview

Python Quiz Application is an interactive quiz platform designed to help users test and improve their Python programming knowledge. The application provides multiple-choice questions, real-time scoring, and a user-friendly interface.

---

## ✨ Features

* 🧠 Python MCQ Questions
* ⏱️ Quiz Timer
* 📊 Instant Score Calculation
* 🎯 Beginner to Advanced Levels
* 🔄 Random Question Selection
* 📱 Responsive Design
* 💾 Score Tracking
* 🎨 Smooth Animations
